# Simon Says Game

A browser-based Simon Says Game built using HTML, CSS, and JavaScript.

## Features

* Random color sequence generation
* User interaction through button clicks
* Level progression system
* Score tracking
* Highest score tracking
* Visual feedback with animations
* Game over detection and restart functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Play

1. Press any key to start the game.
2. Watch the highlighted color sequence.
3. Repeat the sequence by clicking the colored buttons.
4. Each level adds a new color to the sequence.
5. The game ends if you click the wrong color.
